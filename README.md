# ReinforcementLearning
In this Reinforcement Learning it was optimized intravenous fluids and vasopressor dosages for sepsis patients in the ICU. Our model-based approach (tabular Value Iteration) achieved an optimal 82.1% survival rate in a discrete setting, while our continuous QR-DQN model reached 69.3%, outperforming historical human clinician baselines (67.9%).
